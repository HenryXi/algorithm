# Reverse Left Words
[Problem Description](https://leetcode-cn.com/problems/zuo-xuan-zhuan-zi-fu-chuan-lcof/)

```
public String reverseLeftWords(String s, int n) {
    return s.substring(n) + s.substring(0, n);
}
```
